========================================= To connect From MQ Explorer Remote ===============================
To connect to a remote queue manager and perform remote administrative tasks using the WebSphere MQ Explorer, the user executing the WebSphere MQ Explorer is required to have the following authorities: The below commands just allow
================
CONNECT authority on the target queue manager object
INQUIRE authority on the target queue manager object
DISPLAY authority to the target queue manager object  ## setmqaut -m SRC_QM -t qmgr -p haris +dsp +inq +connect
INQUIRE authority to the queue, SYSTEM.MQEXPLORER.REPLY.MODEL
DISPLAY authority to the queue, SYSTEM.MQEXPLORER.REPLY.MODEL  
INPUT (get) authority to the queue, SYSTEM.MQEXPLORER.REPLY.MODEL ## setmqaut -m SRC_QM -n SYSTEM.MQEXPLORER.REPLY.MODEL -t queue -p haris +dsp +inq  +get +put
OUTPUT (put) authority to the queue, SYSTEM.ADMIN.COMMAND.QUEUE 
INQUIRE authority on the queue, SYSTEM.ADMIN.COMMAND.QUEUE  ## setmqaut -m SRC_QM -n SYSTEM.ADMIN.COMMAND.QUEUE -t queue -p haris  +inq +put
Authority to perform the action selected
